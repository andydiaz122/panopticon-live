## Universal Settings (apply to ALL cards)

| Setting | Value |
|---|---|
| Slide size | 1920 × 1080 px (16:9) |
| Background | Pure black `#000000` (NOT #05080F slate — true void) |
| Primary text color | White `#F8FAFC` |
| Accent color | Cyan `#00E5FF` (use sparingly — ONE accent per card max) |
| Whisper color | Cool muted `#5A6678` (chrome / attribution / URLs) |
| Headline font | **Fraunces** (free Google Fonts download) — fallback: system serif (Georgia, Iowan Old Style, Charter, Times New Roman). Anything BUT Helvetica or Arial. |
| Body font | **JetBrains Mono** (free) for any chrome / URL / attribution. Fallback: SF Mono, Menlo. |
| Export | PNG @ 1× (1920×1080), `sRGB` color space, "Best" quality |

**Padding rule:** Editorial generosity. The center text block should occupy ≤ 40% of the slide. The rest is intentional negative space. If your slide feels "balanced and full" — you've added too much.

**The keyboard SFX should END exactly when this card appears** — the typing was the question; the answer arrives in silence. CapCut handles this in the audio mix.

## Build Sequence in Keynote (estimated 25 minutes total)

1. **New Presentation → Custom Size → 1920 × 1080**
2. **Set master slide background to `#000000`**
3. **Install Fraunces font** if not already installed:
   `https://fonts.google.com/specimen/Fraunces` → download → install Fraunces-Regular.ttf, Fraunces-Italic.ttf, Fraunces-Medium.ttf, Fraunces-MediumItalic.ttf
4. **Slide 1 = Card 1**: Insert text box, paste headline + subhead, apply specs from table above. Use Keynote's Format → Text → Character Spacing for letter-spacing.
5. **Duplicate Slide 1 → Slide 2 = Card 2**: Replace text, change color to `#00E5FF`, apply italic.
6. **Duplicate Slide 1 → Slide 3 = Card 3**: Replace with closing layout. For the accent rule, insert a Line shape, set width = 280pt height = 1pt, color cyan @ 70% opacity. Group with text.
7. **Export**: File → Export To → Images → PNG → "All Slides" → 1920×1080 → "Best Quality" → Save to `~/Documents/Panopticon_TitleCards/`

That's it. 3 PNG files. Total file size < 500 KB combined. CapCut imports them in 5 seconds.

## Quality Audit Checklist (run BEFORE handing to CapCut)

For each PNG:

- [ ] Pure `#000000` background (open in Preview → Cmd+Click → "Show Magnifier" → confirm corners are pure black, not slightly gray)
- [ ] Text is crisp at 100% zoom (no jaggies on serif edges)
- [ ] Letter-spacing has been applied (compare to default Keynote rendering — "default Fraunces" is too tight; the spec values look more editorial)
- [ ] Cyan is exactly `#00E5FF` (not the default Apple "Cyan" which is `#0095FF` — that's brand blue, different)
- [ ] No spelling errors (especially Card 1 — "malleable" is commonly misspelled "maleable" or "malleablle")
- [ ] No trailing whitespace inside text boxes (causes alignment drift)

## Why This Card Set Works

- **Three cards, not seven:** every additional title card is a moment the product is NOT on screen. We minimize cards to maximize product time.
- **Pure void backgrounds:** `#000000` (not slate) for the bookends creates the maximum register switch from the dashboard's `#05080F` HUD slate. The bookend register IS the punctuation.
- **Single accent color (cyan only):** judges scan for visual coherence. Two accent colors = noise; one accent color = identity.
- **Italic on Card 2 + Card 3 headline:** italic IS Anthropic's signature emphasis device. Use it deliberately, never decoratively.
- **Cyan accent rule on Card 3:** the only piece of "design" on the closing card. It rhymes with Card 2's cyan italic and the dashboard's cyan SignalBars — three uses of cyan, forming a triadic identity arc across the 3 minutes.

**Ship these. The product carries the rest.**
